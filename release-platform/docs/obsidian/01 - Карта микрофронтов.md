## Общие правила

- Каждый модуль берет настройки через `useSettings()`.
- Модули с внешними API обычно используют `settings.useProxy`, `settings.proxyBase`, `settings.proxyMode`.
- Доступность модулей считается через `legacy/rb_roles_access.json`, `src/services/legacy.ts` и local overrides.
- URL синхронизируется через query `?module=<legacyId>` для совместимости с legacy.
- Sidebar-кнопки могут иметь прямой `href`; внутренние переходы перехватываются adapter-слоем, внешние ссылки остаются обычными ссылками.

## Где искать ответственность

- UI и состояние конкретного экрана: `src/modules/<Module>/index.tsx`.
- Доменные запросы, нормализация и агрегация: `src/services/<domain>.ts`.
- Общие настройки, токены, роли: `src/types/index.ts`, `src/context/SettingsContext.tsx`, `src/services/legacy.ts`.
- Общая навигация: [[UI Navigation Kit]], `src/components/navigation/NavigationSidebar.tsx`, `src/navigation/sidebarMockApi.tsx`.
