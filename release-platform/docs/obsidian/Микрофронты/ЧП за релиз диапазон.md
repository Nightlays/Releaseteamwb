# ЧП за релиз диапазон

## Назначение

Детализирует ЧП по релизам и конкретному стриму за диапазон: задача, название, стрим, сабстрим, кто принес, время, OK/MR merged и tags.

## Где код

- UI: `src/modules/ChpReleaseRange/index.tsx`
- Сбор данных: `src/services/releasePages.ts`
- Реестр: `chpRange` в `src/config/modules.tsx`
- Legacy fallback: `legacy/ЧП за релиз диапазон.html`

## Данные и сервисы

- [[Deploy Lab]]:
  - issues по релизам и платформам.
- [[YouTrack]]:
  - issue metadata и tags.
- [[GitLab]]:
  - MR merged enrichment.
- [[Proxy]]:
  - gateway.

## Настройки

- `deployLabToken`
- `ytToken`
- `gitlabToken`, `gitlabCookie`
- `proxyBase`, `proxyMode`, `useProxy`

## Что показывает

- отдельные вкладки Android/iOS;
- таблицу ЧП для выбранного стрима;
- счетчики по платформам;
- CSV export.

## Что важно при рефакторинге

Модуль использует тот же service family, что [[Анализ релизов за квартал]]. Лучше не дублировать parsing релизов и issue enrichment, а выделить shared `releasePagesDomain`.
