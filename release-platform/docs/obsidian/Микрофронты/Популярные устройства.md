# Популярные устройства

## Назначение

Показывает распределение аудитории по моделям устройств и OS versions для Android/iOS. Используется для понимания, какие устройства важны для регрессии и smoke/sanity.

## Где код

- UI: `src/modules/Devices/index.tsx`
- BI service: `src/services/youtrack.ts`
- Реестр: `devices` в `src/config/modules.tsx`
- Legacy fallback: `legacy/Популярные устройства.html`

## Данные и сервисы

- [[BI]]:
  - device datasource `14517`;
  - OS datasource `16888`;
  - queue/cache API.

## Настройки

- `biCookie`
- `proxyBase`, `proxyMode`, `useProxy`

## Локальное состояние

- `bi_devices_history`

## Что показывает

- top 5/top 10 устройств;
- Android/iOS отдельными таблицами;
- OS breakdown;
- history snapshots;
- delta к предыдущему снимку;
- PDF export.

## Что важно при рефакторинге

Логику BI queue polling лучше вынести в отдельный `biClient`, потому что [[Пользователи по версиям]] и этот модуль решают похожую задачу через разные datasource IDs.
