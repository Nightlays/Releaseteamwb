# Настройки и запуск

## Frontend

```bash
cd release-platform
npm install
npm run dev
```

Vite dev server по умолчанию: `http://localhost:5173`.

Build:

```bash
npm run build
```

## Локальный proxy

```bash
cd release-platform/legacy
node proxy-standalone.js
```

Health:

```bash
curl http://127.0.0.1:8787/health
```

В UI:

- `proxyBase`: `http://localhost:8787`
- `proxyMode`: `query`
- `useProxy`: `true`

## GLM adapter

```bash
cd release-platform
GLM_API_KEY='CORELLM_JWT' node GLM/glm-zai-local-adapter.mjs
```

Health:

```bash
curl http://127.0.0.1:8789/health
```

В UI:

- `glmBase`: `http://localhost:8789/v1`
- `glmModel`: `glm-5.1` или актуальная модель adapter.
- `glmKey`: можно оставить пустым, если adapter запущен с `GLM_API_KEY`.

## ML retrain helper

```bash
cd release-platform/legacy
python3 ml_retrain_helper.py
```

Health:

```bash
curl http://127.0.0.1:8788/health
```

В UI:

- `mlHelperBase`: `http://127.0.0.1:8788`

## Docker legacy stack

В `legacy/docker-compose.yml` описаны:

- `web` - nginx для статического сайта;
- `proxy` - `proxy-standalone.js`;
- `llm` - `GLM/glm-zai-local-adapter.mjs`.

Запуск из `release-platform/legacy`:

```bash
GLM_API_KEY='CORELLM_JWT' docker compose up
```
